## Տեղադրում և Շահագործում

---

### Առանց տեղադրելու

```sh
# Ռեպոզիտորիայի կլոնավորում։
git clone https://github.com/stpatriarch/onekey.git && cd onekey

# Գործարքում։ Այս մեթոդը թույլ է տալիս առանց տեղադրելու այն գործարքե որպես սկրիպտ։ 
python3 src/onekey/core/core.py 

```

### Տեղադրում

```sh
# Ռեպոզիտորիայի կլոնավորում։
git clone https://github.com/stpatriarch/onekey.git && cd onekey

# վիրտուալ աշխատավայրի ստեղծում և ակտիվացում։
python3 -m venv venv && source venv/bin/activate

# տեղադրում։
pip install .
```

### Շահագործում

```sh
# առաջին մեթոդ: Այս առաձին կպահանջի մուտքադրել բանլալին հետո հոսթը։
# Գեներացված գաղտվաբառը կպատճենի փոխանակման բուֆֆերում։
onekey

# երկորորդ մեթոդ։ կարելի է միանգամից տեղդրել և բանալին և հոսթը։
# Գեներացված գաղտվաբառը կպատճենի փոխանակման բուֆֆերում։
onekey -k your_key_here -h your_host_here

# նոր ապահով բանալիի գեներացիա եթե չունեք այն։ Աշխատում է երկու ռեժիմով։ 

# Ուղղակի պատահական գեներացիա։ Ենթակա չէ վերականգնման։
onekey --keygen 

# Գեներացիա ձեր նախընտրած արտահայտությունից։ Կարելի է վերականգնել։
onekey --keygen secret_word 


```
